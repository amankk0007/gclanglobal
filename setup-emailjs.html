Setup Automated Email System

Follow these steps to get 100% automated email sending for your Global Pass Career website.

Important: Choose ONE of the following options. EmailJS is recommended for best results.

Option 1: EmailJS (Recommended - Free & Easy)

Steps:

  1. Go to EmailJS.com
  2. Sign up for free account
  3. Create an Email Service (Gmail recommended)
  4. Create an Email Template
  5. Get your Service ID, Template ID, and Public Key
Template Code:
Subject: Homepage Consultation Request - Global Pass Career

Content:
Hello Global Pass Team,

You have a new consultation request:

Name: {{name}}
Email: {{email}}
Phone: {{phone}}
Interested In: {{message}}
Time: {{timestamp}}

Please contact this person within 24 hours.

Thanks,
Global Pass Website

Then update your code with:

SERVICE_ID = 'your_service_id'
TEMPLATE_ID = 'your_template_id'
PUBLIC_KEY = 'your_public_key'

Option 2: Web3Forms (Alternative - Free)

Steps:

  1. Go to Web3Forms.com
  2. Sign up for free account
  3. Create a new form
  4. Add your email: info@globalpasscareer.com, amankk0007@gmail.com
  5. Get your Access Key
Update your code with:
WEB3FORMS_ACCESS_KEY = 'your_access_key'

Option 3: Formspree (Alternative - Free)

Steps:

  1. Go to Formspree.io
  2. Sign up for free (Gold plan for multiple recipients)
  3. Create a new form
  4. Configure to send to both your emails
  5. Get your Form ID
Update your code with:
FORMSPREE_ENDPOINT = 'https://formspree.io/f/your_form_id'

After Setup:

Once you configure any of these services:

  • Forms will send emails automatically
  • No user intervention required
  • Works directly from browser
  • Multiple fallback options included
  • Professional email formatting

Add EmailJS Script to Your Website

Add this script to your main HTML file (index.html or similar):

<script type="text/javascript" src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>
<script type="text/javascript">
  (function() {
    emailjs.init("YOUR_PUBLIC_KEY");
  })();
</script>
